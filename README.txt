To open:
- Double-click the fireworksProgram.html file and the program will open in browser
- Recommended to open with Google Chrome browser for best performance

To Use the Program
- Unless auto-launching is enabled, click the black background to launch a firework
- Adjust the features in the user settings by click the on/off switch and using the +/- buttons
*NOTE: Numbers in the textbox are for display only, do not input numbers there, only use +/- buttons to adjust the features*

Basic fireworks launch program from: https://github.com/paullewis/Fireworks
